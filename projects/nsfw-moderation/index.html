NSFW Moderation Pipeline - Image Content Filtering

Developed NSFW content moderation pipeline using NudeNet and MobileNet with 97% accuracy and 180ms inference time through advanced optimization.
September 1, 2024 · 1 min

Overview#

Built a comprehensive NSFW content moderation pipeline for the Dip social media app, achieving high accuracy with optimized performance.

Pipeline Architecture#

Dual-Model Approach#

  1. NudeNet Detection Model: Primary NSFW detection
  2. MobileNet Classifier: Secondary classification and validation

Dataset#

  • Source: Custom NSFW scraper
  • Size: 12 GB of diverse NSFW content
  • Classes: Multiple NSFW categories for fine-grained classification

Performance Optimization#

Challenge#

  • Initial Performance: 250ms inference time (CPU)
  • Target: Real-time moderation (<200ms)
  • Benchmark: Ultralytics’ own performance standards

Optimization Strategy#

1. Model Conversion#

  • Converted PyTorch → ONNX format
  • ONNX → OpenVINO using Model Optimizer (MO)

2. Post-Training Optimization (POT)#

  • Applied OpenVINO Post-Training Optimization
  • Quantization for faster inference
  • Graph optimization techniques

Results#

  • Inference Time: 250ms → 180ms
  • Performance Gain: 28% improvement
  • Accuracy: 97%
  • Beat: Ultralytics’ own benchmarks

Pipeline Performance#

  • Latency: 180ms per image
  • Accuracy: 97%
  • Real-time Processing: Yes
  • Scalability: High
